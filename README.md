# Mingyue Sun — Portfolio

Frontend developer portfolio built from scratch with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools — just clean code and thoughtful design.

[View Live Site](https://mingyue-sun-dev.github.io/portfolio)

## About This Project

This portfolio was designed and built with vanilla web technologies — no frameworks or build tools. Animations, interactions, and layout are all implemented in pure HTML, CSS, and JavaScript.

### Highlights

- **Custom cursor system** with smooth trailing animation
- **3D tilt effect** on project cards using perspective transforms
- **Magnetic hover buttons** that follow cursor movement
- **Animated text reveal** with per-word staggered 3D entrance
- **Scroll-based parallax** on hero elements
- **Infinite marquee ticker** for skills (pauses on hover)
- **Light/dark theme toggle** with system preference detection
- **Animated gradient background** with floating orbs and geometric shapes
- **Fully responsive** across desktop, tablet, and mobile
- **Accessibility-aware** — respects `prefers-reduced-motion`

## Built With

- **HTML5** — semantic, accessible markup
- **CSS3** — custom properties, keyframes, grid, flexbox, `backdrop-filter`
- **JavaScript (ES6+)** — modular IIFE architecture, IntersectionObserver, requestAnimationFrame
- **EmailJS** — contact form integration
- **Google Fonts** — Outfit (headings) + Inter (body)

## Run Locally

```bash
python3 -m http.server 8000
```

No dependencies to install. Just open and go.
